## zynq-library (2018.2)
Various Linux library files for Zynq-7000 series
  
***
#### arm

- ARM ComputeLibrary & NN SDK

  - https://developer.arm.com/technologies/compute-library
  - https://developer.arm.com/products/processors/machine-learning/arm-nn

- ffmpeg (4.0.1)
